## 📌 Overview

This repository provides automated shell scripts to deploy the RoboShop microservices application.  
It simplifies the manual setup process by configuring all services, databases, and dependencies using Bash scripting on Linux systems.

---

## 🧩 Features

- One-command deployment of RoboShop application  
- Modular scripts for each microservice  
- Database setup (MongoDB, MySQL, Redis)  
- Messaging queue setup (RabbitMQ)  
- Nginx frontend configuration  
- Reusable common script  

---

## 🛠️ Tech Stack

- Shell Scripting (Bash)
- Linux (CentOS / Ubuntu)
- Nginx
- MongoDB
- MySQL
- Redis
- RabbitMQ
